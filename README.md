# EstudioJuridicoMC

Landing page desarrollada para un cliente del rubro legal.

## Contexto

Este proyecto corresponde a un sitio web de presentación para un estudio jurídico. El objetivo principal es comunicar servicios, transmitir confianza profesional y facilitar el contacto con potenciales clientes.

## Objetivo de la landing

- Presentar la propuesta de valor del estudio jurídico.
- Mostrar servicios de forma clara y ordenada.
- Guiar al usuario hacia una acción de contacto.

## Stack

- Vite
- JavaScript
- SCSS

## Estado

Proyecto orientado a despliegue web como landing page pública para cliente.
